# Architecture Decision Records (ADR)

This folder holds ADRs produced by `banking-solution-architect` and `banking-tech-lead` agents.

## Format

Use MADR (Markdown ADR) format. File naming: `NNNN-kebab-title.md`.

## Template

```markdown
# NNNN. <Decision Title>

- **Status:** proposed | accepted | deprecated | superseded by NNNN
- **Date:** YYYY-MM-DD
- **Deciders:** <agent names>
- **Tags:** <area>

## Context and Problem

<What is the issue we're trying to solve?>

## Decision Drivers

- Driver 1
- Driver 2

## Considered Options

1. Option A
2. Option B
3. Option C

## Decision Outcome

Chosen: **Option X** because <rationale>.

### Consequences

- ✅ Positive consequence
- ⚠️ Negative consequence / trade-off

## Links

- [Related ADR](./NNNN-other.md)
- [External reference](https://...)
```
